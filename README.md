## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`




# MK AI OS – Personal AI Operating System

MK AI OS is a full-stack AI-powered personal operating system that combines an intelligent web dashboard, AI assistant, Telegram integration, and a desktop companion into one unified platform.

Designed and built as a personal software engineering project, MK AI OS demonstrates how modern AI services, cloud technologies, and desktop applications can work together to create a seamless user experience.

---

## 🚀 Features

### 🤖 AI Assistant
- Google Gemini powered conversational AI
- Intelligent question answering
- Context-aware interactions

### 🔐 Authentication
- Firebase Authentication
- Secure user login and registration
- Multi-user support

### 🖥️ Personal Dashboard
- Modern responsive UI
- User profile management
- AI interaction dashboard

### 💻 Laptop Companion Agent
A Python desktop agent that securely communicates with the cloud.

Features include:
- Real-time CPU monitoring
- RAM usage monitoring
- Disk usage monitoring
- Running process information
- Desktop-to-cloud synchronization
- Remote command execution

### 📱 Telegram Integration
- Telegram Bot integration
- Send commands remotely
- Receive AI responses
- Control supported desktop functions

### 📧 Gmail Integration
- Gmail connectivity
- Email-related features within the platform

### 📅 Productivity
- Task management
- Calendar support
- File handling

### ☁ Cloud Deployment
- Deployed on Vercel
- Cloud-based backend services
- Progressive Web App (PWA)

# 🔄 Remote Communication

MK AI OS enables communication between:

```
User
   │
   ▼
Web Dashboard
   │
   ▼
Express Server
   │
   ▼
Laptop Companion Agent
```

The desktop companion receives supported commands from the backend and reports system information back to the dashboard.


```bash

```

## Start Development Server

